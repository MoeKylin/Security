## 下载安装

- Debian/Ubuntu
```
apt-get update
apt-get install whatweb
```
- Centos
```
yum update
yum install ruby ruby-devel rubygems
wget http://www.morningstarsecurity.com/downloads/whatweb-0.4.7.tar.gz
tar xzvf whatweb-0.4.7.tar.gz
```
